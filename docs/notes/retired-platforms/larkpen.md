# LarkPen — Retired

**Decision date:** 2026-06-06
**Evidence:** HTTP 200 all UAs. `site:larkpen.com` returns only the homepage. No user-generated content pages indexed in Google. Cannot find a working user blog post URL to inspect for dofollow status.
**Decision rationale:** Platform has no discoverable user content. Google has not indexed any post pages. Even if dofollow were hypothetically possible, the SEO value is negligible with no indexation. Not worth adapter investment.
**Reopen conditions:** If LarkPen gains traction and user content becomes indexable (10+ indexed user pages). Check when: re-run `site:larkpen.com` in 6 months.
